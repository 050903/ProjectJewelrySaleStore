# Cửa hàng trang sức

Chào mừng bạn đến với cửa hàng trang sức của chúng tôi! 
## Phân Loại:
- Chất lượng cao
- Mẫu mã độc đáo
- Nhập khẩu quốc tế

## Cài đặt:
1. Clone the repository.
2. Mở thư mục dự án
3. Kích hoạt môi trường ảo
4. Chạy python manage.py runserver
